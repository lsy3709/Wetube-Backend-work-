# 01. 작업 개요

## 목적

Flask WeTube(ch05)에 **비디오·썸네일 업로드 기능**을 추가한 작업 내역을 문서화한다.

## 구현 범위

- **페이지**: `/studio/upload`
- **폼 입력**: 제목(필수), 설명, 비디오 파일(필수), 썸네일(선택)
- **저장 위치**: config의 `VIDEO_FOLDER`, `THUMBNAIL_FOLDER`
- **DB**: Video 모델에 메타데이터 저장 (SQLite)
- **검증**: 허용 확장자·최대 용량 초과 시 에러 메시지 표시 후 폼 다시 표시

## 참고

- 로그인 미연동 상태이므로 업로드 시 `config.DEFAULT_USER_ID`(기본 1) 사용.
- DB에 `user_id=1` 사용자가 없으면 앱 기동 시 `username=default` 사용자를 자동 생성.
